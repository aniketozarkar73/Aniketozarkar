Q.11• What is the difference between a block-level element and an inline element?  
Ans-1.Block-level Elements:
->Always start on a new line.
->Automatically add space (margins) before and after the element.
->Take up the full width available (stretch from left to right).
->Common examples: <p>, <div>, <h1> to <h6>, <section>, and <article>

2.Inline Elements:
->Do not start on a new line; they flow within the same line.
->Only take up as much width as necessary.
->Ideal for applying specific styles or adding interactive elements.
Examples include <span>, <a>, <strong>, <em>, and <img>1.

Q.13 What is the use of an iframe tag?
Ans-The <iframe> tag in HTML is used to embed another document within the current webpage. It acts like a window, displaying external content such as videos, maps, or social media feeds without redirecting the user to another page.

Q.14 What is the use of a span tag? Explain with example?
Ans-The <span> tag in HTML is used to apply inline styling to a specific portion of text within a larger block of content. It doesn’t create any visual change by itself but allows you to target and style specific parts of the text.
ex-
->The word “blue” will be displayed in blue color because of the <span> tag.
->The <span> tag acts as a container for the styled text.

Q.15 How to insert a picture into a background image of a web page?

Q.16 How are active links different from normal links? 
Ans-1.Normal Links (Unvisited Links):
->A normal link is a simple line of code that contains a pointer to another resource (usually another web page).
->When you create a link using the <a> tag in HTML, it represents a normal link.

2.Active Links:
An active link is a hyperlink that is currently being interacted with by the user.
When you click (left or right-click) on either a visited or unvisited link, it becomes an active link.

Q.17 What are the different tags to separate sections of text? 
Ans- 1. Paragraph (<p>)
The <p> tag is used to define paragraphs. It automatically adds space before and after the paragraph
2. Line Break (<br>)
The <br> tag inserts a line break within text, moving the content that follows to a new line without starting a new paragraph.
3. Horizontal Rule (<hr>)
The <hr> tag creates a horizontal line that can be used to separate content sections visually.
4. Division (<div>)
The <div> tag is a block-level container used to group and separate sections of content. It is commonly used with CSS for styling and layout purposes.

Q.18 What is SVG? 
Ans-SVG (Scalable Vector Graphics) is an XML-based markup language for describing two-dimensional vector graphics.

Q.19 What is difference between HTML and XHTML? 
Ans-                                                     
HTML->
-HTML stands for Hypertext Markup Language
-It is an SGML application
-HTML is not case sensitive
-HTML can use open tags, such as <br>
-HTML is less expressive.


XHTML->
-XHTML stands for Extensible Hypertext Markup Language
-It is an XML application
-XHTML is case sensitive
-All unclosed tags must be closed in XHTML
-XHTML is more expressive as compared to HTML
